# Daily Grok Instructions

Put your instructions for opencode here. This file gets executed once daily at 9am.

## Format

Write plain English instructions. The entire file content is passed as a prompt to opencode (deepseek-v4-flash-free). It can use any tool available in the MIDIS_ALL_REAL toolchain.

## Example

```
Run 47_propagator.py then 48_active_pool.py, respecting resumability. Report only the md5s processed.
```

## Another example

```
Run CODE/49_sig_one.py --midi /some/path.mid --compare and report the top 5 nearest neighbors.
```

## Notes

- This file is archived to `_dailygrok_archive/` after execution
- Changes get committed and pushed back automatically
- The script runs from the repo root with all tools available
